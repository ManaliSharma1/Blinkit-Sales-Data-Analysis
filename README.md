# Blinkit Sales Data Analysis

## Project Overview

This project analyzes Blinkit sales data using SQL to uncover insights related to sales performance, customer behavior, and outlet characteristics. The analysis includes data cleaning, KPI computation, and advanced SQL operations.

## Business Objective

The objective of this project is to analyze sales data and generate actionable insights to improve business performance, optimize product strategy, and understand customer preferences.

## Data Cleaning

Standardized inconsistent values in the Item_Fat_Content column to ensure data consistency and improve the accuracy of analysis.

## Key Performance Indicators

* Total Sales
* Average Sales
* Number of Orders
* Average Rating

## Analysis Performed

### Sales Analysis

Analyzed total sales by item type and fat content to identify high-performing product categories.

### Outlet Analysis

Evaluated sales performance across outlet locations, sizes, and establishment years to identify key revenue drivers.

### Product Insights

Examined item type performance and customer ratings to understand product demand and customer preferences.

### Advanced SQL Techniques

Used aggregations, window functions, and pivot operations to perform detailed analysis and generate insights.

## Tools and Technologies

SQL (MS SQL Server)

## Key Insights

* Sales vary significantly across different item types and outlet categories
* Outlet size and location influence overall revenue
* Customer ratings vary across product categories and outlet types

## Conclusion

This project demonstrates the use of SQL for data cleaning, transformation, and analysis to generate business insights and support decision-making.
